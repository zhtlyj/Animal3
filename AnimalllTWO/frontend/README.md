# 动物保护平台 - 前端应用

基于React的动物保护公益与领养平台前端用户界面。

## 功能特性

- 🔐 用户注册和登录系统
- 👥 四种用户身份类型：游客、救助组织、领养人、捐赠者
- 🎨 现代化、响应式的用户界面
- 🔒 受保护的路由和状态管理
- 📱 移动端友好的设计

## 用户身份类型

1. **游客** - 可以浏览平台所有功能，参与公益活动
2. **救助组织** - 可以发布动物救助信息，管理救助动物档案，接收捐赠
3. **领养人** - 可以浏览可领养动物信息，提交领养申请
4. **捐赠者** - 可以查看救助项目，进行爱心捐赠

## 技术栈

- React 18
- React Router DOM
- CSS3 (现代渐变和动画效果)
- MongoDB (后端数据存储)
- RESTful API (前后端通信)

## 安装和运行

### 从项目根目录启动
```bash
# 在项目根目录
npm run start:frontend
```

### 单独启动前端
```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm start
```

访问地址：http://localhost:3000

## 使用说明

### 注册新用户
1. 访问注册页面
2. 选择用户身份类型
3. 填写用户ID、电话号码、邮箱
4. 设置密码并确认
5. 点击注册按钮

### 用户登录
1. 访问登录页面
2. 选择用户身份类型
3. 输入用户ID和密码
4. 点击登录按钮

### 用户仪表板
登录成功后，用户将看到：
- 个人用户信息
- 根据身份类型显示的功能权限
- 退出登录选项

## 开发说明

- 用户数据存储在浏览器的LocalStorage中
- 密码验证功能已预留接口，实际部署时需要连接后端API
- 所有表单都有客户端验证
- 支持响应式设计，适配移动端和桌面端
